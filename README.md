# MyCalculator

my first project of calulator

## 📋 Description

MyCalculator is a simple Python calculator application that supports basic arithmetic operations. It's a great beginner project for learning Python and GitHub!

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero-division protection)
- 🔋 Power (exponentiation)
- User-friendly command-line interface
- Input validation
